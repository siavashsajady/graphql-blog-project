# GraphCMS Blog

## Introduction

Ù‘Fully responsive CMS Blog App with React, GraphQL, NextJS, Tailwind CSS, and features such as recent posts, categories, full markdown articles, author information, comments. Users can manage the blog from a dedicated Content Management System.

## Usage

To run the project

1. clone or download this repo `gh repo clone siavashsajady/graphql-blog-project`
2. Install dependencies `npm install`
3. Run the project using `npm run dev`
